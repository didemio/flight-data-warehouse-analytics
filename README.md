# Flight Operations Data Warehouse & Analytics System

## Overview
This project implements a data warehouse for airline operations analytics using Oracle SQL.

## Features
- Star schema design (fact + dimension tables)
- ETL process using PL/SQL
- Flight performance analysis (delay, profit, load factor)
- Advanced SQL (ROLLUP, CUBE, GROUPING SETS)
- What-if analysis using MODEL clause

## Technologies
- Oracle SQL
- PL/SQL
- Data Warehousing
- OLAP Queries

## Structure
- `/sql` → ETL and warehouse scripts
- `report.pdf` → Full project report

## Key Insights
- Route profitability analysis
- Delay and on-time performance
- Load factor trends
- Fuel cost impact analysis
